メラメラしたりぼこぼこしたり

googlefonts

何をするかって言うとメラメラしたりぼこぼこしたりするってことです。

google fonts のドキュメントを漁っていたら、font effects なるものを見つけました。 β 版なものの 簡単に font にスタイルを与えられていろいろ遊べそうです。

導入方法

ここに effect のリストがあるので好きな effect を選んで、google fonts と同じようにeffect=<effect_name>で指定してfont-effect-<effect_name>を class に適用するだけです。

<head>
  <link
    rel="stylesheet"
    href="https://fonts.googleapis.com/css?family=Roboto&effect=fire"
  />
</head>
<body>
  <h1 class="font-effect-anaglyph">こんにちは</h1>
</body>
