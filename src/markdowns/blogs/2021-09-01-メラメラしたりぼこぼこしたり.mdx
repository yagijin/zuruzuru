何をするかって言うと<span class="font-effect-fire-animation">メラメラ</span>したり<span class="font-effect-emboss">ぼこぼこ</span>したりするってことです。

google fonts のドキュメントを漁っていたら、[font effects](https://developers.google.com/fonts/docs/getting_started#enabling_font_effects_beta)
なるものを見つけました。 β 版なものの 簡単に font にスタイルを与えられていろいろ遊べそうです。

## 導入方法

[ここ](https://developers.google.com/fonts/docs/getting_started#enabling_font_effects_beta)に effect のリストがあるので好きな effect を選んで、google fonts と同じように`effect=<effect_name>`で指定して`font-effect-<effect_name>`を class に適用するだけです。

```html
<head>
  <link
    rel="stylesheet"
    href="https://fonts.googleapis.com/css?family=Roboto&effect=fire"
  />
</head>
<body>
  <h1 class="font-effect-anaglyph">こんにちは</h1>
</body>
```
